## 🗂️ Categorias do inventário

### 🔧 Ferramentas manuais
Alicates, chaves de fenda, estiletes, réguas, tesouras, martelos, etc.

### ⚡ Ferramentas elétricas
Furadeira, parafusadeira, ferro de solda, pistola de cola quente, etc.

### 💻 Equipamentos digitais
Computadores, notebooks, tablets, projetores, impressoras 3D, etc.

### 🤖 Kits de robótica e eletrônica
Arduino, micro:bit, sensores, motores, protoboards, componentes diversos.

### 📐 Materiais de prototipagem
Papelão, MDF, EVA, isopor, materiais recicláveis, filamento PLA, etc.

### 🛡️ EPIs (Equipamentos de Proteção Individual)
Óculos de proteção, luvas, máscaras, aventais.

### 📚 Materiais didáticos
Livros, manuais, kits temáticos, jogos educativos.

## 🏷️ Padrão de cadastro

Cada item do inventário deve conter:

| Campo | Exemplo |
|-------|---------|
| **Código** | `FER-001`, `EQP-012`, `KIT-005` |
| **Nome** | Furadeira Bosch GSB 550 |
| **Categoria** | Ferramenta elétrica |
| **Quantidade** | 2 unidades |
| **Estado** | ✅ Bom / ⚠️ Regular / 🔧 Manutenção / ❌ Inutilizado |
| **Localização** | Armário A, prateleira 2 |
| **Data de aquisição** | 03/2026 |
| **Origem** | SED/SC, doação, recurso próprio |
| **Observações** | Manual disponível em `recursos/` |

## 🔄 Fluxo de empréstimo

```
1. Professor solicita item → preenche formulário
2. Orientador Maker verifica disponibilidade
3. Registro em emprestimos.md (data, responsável, prazo)
4. Devolução conferida → atualização do estado
5. Se houver dano → registrar em manutencao.md
```

## 🛡️ Boas práticas

- ✅ **Atualizar imediatamente** após qualquer movimentação
- ✅ Realizar **auditoria mensal** dos itens
- ✅ Manter **manuais e notas fiscais** arquivados
- ✅ Etiquetar fisicamente todos os itens com o **código do inventário**
- ✅ Registrar **fotos** de itens de alto valor
- ✅ Comunicar **perdas ou danos** à gestão escolar

---

## 🔐 Segurança e responsabilidade

- O uso de ferramentas elétricas e cortantes **requer supervisão**
- EPIs são **obrigatórios** em atividades de risco
- Itens de alto valor ficam em **armário trancado**
- Empréstimos exigem **assinatura do responsável**
